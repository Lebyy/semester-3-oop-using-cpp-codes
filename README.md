## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- A C++ compiler
- A text editor

### Installation

1. Clone the repo

```sh
git clone Lebyy/semester-3-oop-using-cpp-codes
```

2. Compile the code

```sh
g++ -o <output_file_name> <input_file_name>
```
